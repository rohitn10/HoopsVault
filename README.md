# HoopsVault

A dynamic NBA trading card marketplace prototype prepared for Vercel + Supabase.

## Run locally

From this folder, start a local server:

```bash
python -m http.server 3000
```

Then open http://localhost:3000 in your browser.

## Next steps for Vercel + Supabase

1. Create a Supabase project.
2. Enable Email and Google authentication.
3. Create `listings` and `messages` tables.
4. Replace the placeholder values in `config.js` with your Supabase URL and anon key.
5. Deploy the site to Vercel.
